# Twig

Minimal-effort reasonably preconfigured logger for Python.

## Installation

```{shell}
python -m pip install git+https://github.com/numpde/twig.git@4d2b52
```

## Usage

```python
from twig import log, LOG_FILE
log.info("Hello, Dave.")
log.info(LOG_FILE)
```
